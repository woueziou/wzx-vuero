| Props            | Default                                       | Type                                              |
| ---------------- | --------------------------------------------- | ------------------------------------------------- |
| trueValue<br />  | <span class="is-undefined">`undefined`</span> | any                                               |

| modelValue       | <span class="is-array">`[]`</span>            | any                                               |
